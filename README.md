# Sat-Hesaplama
Ticaret için gerekli hesaplamaların yapılabileceği ürün bilgisine göre ana para hedef para hesaplamaya yarayan uygulama
Alınan ürün adedi fiyatı ve ana para bilgisine göre hedef para hesaplama
Hedeflenen X Tl için Y TL sermaye ile ürün bilgisi hesaplama
X TL ana para ve belirli ürünler üzerinden hedef para hesaplama
